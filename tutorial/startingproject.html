Starting the Project
  1. Start up Project Builder (/Developer/Applications/Project Builder).
  2. Go to File -> New Project.
  3. Select a Cocoa Application (not a Document-based Application). Document-based applications are reserved for applications which open up multiple instances of its workspace window, such as a text editor. Since the EdenMath calculator has need for only one window, this project should be just a standard Cocoa Application.
  4. Name the program whatever you want. In this example, it has been named EdenMath.
